# SlideZ
Last minute slides generation using Gen Ai

SlideZ is an open-source GenAI engine that converts raw ideas into fully structured slide decks programmatically. Designed for extensibility, customization, and community-driven innovation, it enables users and developers to adapt and enhance the system for diverse use cases.

This project focuses on building an automated presentation generator by our open-source Generative AI models. Users provide a topic, keywords, or a short description, and the system automatically generates a structured, well-formatted slide deck with logically organized content and smooth narrative flow.

By leveraging AI-driven automation, SlideZ streamlines presentation creation by intelligently handling content generation, slide organization, and overall coherence—significantly reducing manual effort and saving time.
